מבני נתונים

למשנים ניתן לתת ערכים מטיפוסים שונים. הטיפוס הבסיסי ביותר הוא מחרוזות, רצף של תווים בין גרשיים כמו לדוגמא
"John", "I'm 32 years old" או "www.google.com" ו מספרים.

שני הטיפוסים הללו בנוסף לטיפוס בוליאנים (true או false) ו Null and Undefined, ידועים בתור טיפוסים פרימיטיביים. בנוסף לטיפוסים הפרימיטיביים ישנם עוד שני סוגי טיפוסים שהם מערכים ו אובייקטים. נסביר עליהם בהרחבה בהמשך.

כדי לבדוק מה הטיפוס של משתנה אפשר להשתמש באופרטור typeof: 

                    
    var firstName = "John";  
    
    console.log( typeof firstName ); // the result will be "string"

    var age = 40;  
    
    console.log( typeof age ); // the result will be "number"

מחרוזות

כמו בשפות קודמות, מחרוזות הן רצף של תווים. בעצם טקסט. מחרוזות מזוהות באמצעות גרשיים או גרש בודד.

כמה דוגמאות למשתנים עם השמות של מחרוזות: 

                    
    var firstName = "John";  // we can use double quotation marks
    
    var lastName = 'Doe'; // or single quotation marks
    
    var passportNumber = 'XYZ0123';  
    
    var phoneNumber = "909299999";
    
    var email = 'john_doe@gmail.com';
    
    var bio = "John Doe started his web developer career in 2020 and has contributed to many important projects, like...";

נשים לב כי בדוגמא לעיל במשתנה phoneNumber יש ערך של מספרים בלבד. אם כך, למה השתמשנו במחרוזות ולא במספרים?

למרות שהערך הוא מספרים בלבד, לא נשתמש בערך לחישובים מתמטיים ולכן שימוש במחרוזות ישרת אותנו טוב יותר.

מצד שני, אם נרצה לאחסן שנת לידה של מישהו כנראה שנעדיף טיפוס מספרים, כיוון שנרצה לחשב הפרש בין השנה הנוכחית לשנת הלידה שלו כדי לגלות את גילו.

ישנן כמה פעולות חשובות שניתן לעשות עם מחרוזות, כנראה למטה:

  • שרשור
                    
    var firstName = "John";

    var lastName = "Doe";

    var fullName = firstName + " " + lastName;

    console.log(fullName); 

    // The result will be "John Doe"

    // Therefore the plus sign (+) is used for concatenation with string values.
  • אורך מחרוזת ואינדקס

כיוון שמחרוזות הן רצף תווים, יש להן אורך. לכל תו יש מיקום מסויים ברצף, הנקרא אינדקס. 

                    
    var firstName = "John";

    console.log(firstName.length); // The result will be 4

    /* 
    The length property of a string represents its number of characters.
    Each character has its own index, starting from zero.
    White spaces also count. 
    */

    console.log(firstName[0]); // The result will be 'J'
    

    // We can access each character of a string using their index inside brackets.
  • String replace

ישנן דרכים שימושיות לשימוש במתודה replace. נאמר כי יש לנו את הקישור: "https://www.youtube.com". בקלות נוכל להסיר את //:https בהחלפתו עם מחרוזת ריקה. בשביל להשתמש במתודה תחילה נכתוב את החלק אותו נרצה להסיר ואז את הטקסט איתו נרצה להחליף. 

                    
    var url = "https://www.youtube.com";

    console.log( url.replace( "https://" , "" ) ); // It will show "www.youtube.com"

ישנן עוד פעולות רבות שאפשר לבצע על מחרוזות. אם תרצו לחקור עצמאית תוכלו לעשות זאת בקישור.

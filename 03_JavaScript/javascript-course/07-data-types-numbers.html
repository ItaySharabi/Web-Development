מספרים

מספרים מושמים למשתנים ללא גרשיים: 

                    
    var birthYear = 1982; 
    var price = 16.99;  // decimal places must be separated by a dot.
    var temperature = -5;
    
    var bigNumber = 2e45; 
    /* large numbers in Javascript can be represented by the exponential notation. 
    // This example means 2 x 1045 */

לכל חישוב מתמטי ניתן להשתמש במספרים, החל מפעולות חשבון פשוטות וכלה בביטויים מתמטיים.

  • פעולות חשבון פשוטות
                    
    var num1 = 20;
    var num2 = 3;

    var sum = num1 + num2;
    var subtraction = num1 - num2;
    var division = num1 / num2;
    var multiplication = num1 * num2;

    var average = (num1 + num2) / 2;

    /*
    Parenthesis can be used to form expressions. In this case it was used to define the order
    of the operations. Without parenthesis, the division would have been done before the sum,
    resulting in an incorrect average.
    */
  • האובייקט Math

ניתן לבצע פעולות מתקדמות יותר בעזרת האובייקט Math (בו השתמשנו גם בJava). אזכיר שפונקציות של אובייקט נקראות מתודות. כדי לגשת למתודות של אובייקט, יש לכתוב את שמו ולאחריו נקודה. לאחר הנקודה נציין את שם המתודה או השדה של אותו אובייקט. 

                    
    var power = Math.pow(2,4);  /*  This does 24, which is 16. */

    var roundedNumber = Math.round(2.7); // Rounds to the nearest integer. Result: 3

    var roundUp = Math.ceil(4.3) // Rounds to the next integer. Result: 5
    
    var roundDown = Math.floor(4.6) // Rounds to the current integer. Result: 4

    var squareRoot = Math.sqrt(36) // Square root. Result: 6

    var randomNumber = Math.random() // The result is a random number between 0 and 1
  • מתודות של האובייקט מספר

בJavascript, כל טיפוס הוא אובייקט עם מתודות משלו.

כאשר יש לנו משתנה עם השמה למספר, נוכל להשתמש בכמה מתודות: 

                    
    var myNumber = 3.78633;
    
    // toFixed() rounds it to a fixed number of decimal places

    console.log( myNumber.toFixed(2) ); // will show 3.79
    console.log( myNumber.toFixed(4) ); // will show 3.7863

עוד מתודות של מספרים בקישור

  • אופרטור הגדלה

זוג פעולות שכיח שאנו בעיקר משתמשים בו זה הגדלה והפחתה של מספרים. כאשר נתחיל לעבוד עם לולאות זה יהיה הכרחי.

בשביל להוסיף 5 למשתנה מטיפוס מספר ישנן שתי דרכים. 

                    

    var myNumber = 20;
    myNumber = myNumber + 5;
    console.log(myNumber); // The result will be 25

במקום המבנה הנ"ל, נוכל להשתמש באופרטור =+ 

                    

    myNumber = 1;
    myNumber += 5;
    console.log(myNumber); // The result will be 6

    // We can also use *=, /= e -= for incremental multiplication, division and subtraction

האופרטור =+ עובד גם על מחרוזות כשרשור: 

                    
    var name = "John";
    name += " ";
    name += "Doe";

    console.log(name); // The console will show "John Doe"

המרת טיפוסים

יהיו מקרים בהם נצטרך להמיר מטיפוס אחד לאחר. למשל במקרה בו נקבל מספר כמחרוזת (מקרה שכיח). בשביל לבצע על המספר פעולות מתמטיות, קודם נצטרך להמיר אותו לטיפוס מספר. 

                    
    var firstNumber = "21"; 
    var secondNumber = "4";

    console.log(firstNumber + secondNumber);
    // The result will be 214 because the plus sign does concatenation on strings.

כדי להמיר מחרוזות למספרים נשתמש בפונקציות parseInt בשביל מספר שלם וparseFloat בשביל מספר עשרוני. 

                    
   
    var firstNumber = "21"; 
    var secondNumber = "4";

    console.log(  parseInt(firstNumber) +  parseInt(secondNumber) );
    //The result will be 25

    // To do the opposite we can use the toString method.

    var yearOfBirth = 1986;
    
    console.log(yearOfBirth.toString() +  " is the year I was born.");

שימו לב להבדל בין המבנים. להמרת מספר ממחרוזת, נציין את שם הפונקציה שתקבל כארגומנט את המשתנה. להמרת מחרוזת ממספר, אנו בעצם משתמשים במתודה של מספרים, ולכן קודם נכתוב את שם המשתנה נקודה ואת שם המתודה עם סוגריים ריקים.

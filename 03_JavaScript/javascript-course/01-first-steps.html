למה Javascript?

לאחר שלמדנו HTML ו CSS נוכל לבנות אתרים מושקעים. אבל ישנה בעיה - הם סטטים. אין שום אינטרקציה עם המשתמשים.

כאשר בונים אתר עם שימוש בHTML ובCSS בלבד, המשתמשים יכולים לגלול למעלה ולמטה וללחוץ על קישורים. באמצעות Javascript אנחנו יכולים להוסיף שכבה של אינטרקציה אשר תהפוך את האתר שלנו להרבה יותר עשיר במונחים של שימושיות.

משתמשים יוכלו לפתוח ולסגור תפריטים, להרחיב ולצמצם חלקים מסויימים, לנווט דרך סלייד של תמונות, להשתמש בפילטרים והרבה יותר. עם Javascript, האתר שלנו יהיה דינאמי!

איפה לכתוב קוד של Javascript?

קוד של Javascript יכול להכתב בתגיות. התגיות נראות כמו של HTML ויכולות להכתב בhead או בbody. ברוב במקרים המקום הטוב ביותר לכתוב את התגיות של הסקריפט יהיה בסוף תגית הbody. 

                   
    <!DOCTYPE HTML>
    
    <html>

        <head>
            <script>

                // Javascript code can be written here

            </script>
        </head>
        
        <body>


            <!-- page content  -->


            <script>

                // Javascript code can be written here

            </script>

        </body>
    </html>

עם זאת, המקום הטוב ביותר לכתוב את הקוד של Javascript הוא בקובץ js נפרד. במקרה זה, תגית הסקריפט תשרת כקישור לקובץ האמור. 

                    
    <body>


        <!-- the html page  -->


        <script src='js/scripts.js'></script>
        

    </body>

כתיבת הJavascript statements הראשונים

באמצעות קובץ הscripts.js שבדיוק יצרנו, נכתוב כמה statements. 

                    
    alert('Sending a message via alert');

    console.log('Sending a message via the console');

כעת תרעננו את הדף ותראו מה קורה. ההודעה שמועברת דרך פונקציית הalert מופיעה בתוך הודעה קופצת. ההודעה שנשלחה דרך הconsole.log יכולה להיראות בבדיקה של הדפדפן (inspector).

הדברים שאנחנו שולחים לconsole הם לא עבור המשתמש, למעשה זהו כלי מאוד שימושי לזיהוי טעויות בקוד והבנה מה הקוד שלנו עושה.

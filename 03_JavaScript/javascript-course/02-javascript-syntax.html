Javascript Syntax

Statements

Statements הם הוראות שהדפדפן מריץ. כל statement צריך להסתיים עם ; הקוד בJavascript יריץ את המשפטים לפי הסדר, מלמעלה למטה. 

                    
    console.log('statement 1');
    console.log('statement 2');

לכל דפדפן יש את הJavascript interpreter שלו שאחראי להריץ את המשפטים שאנו כותבים. הם גם נקראים מנועי Javascript. בטבלה למטה אפשר לראות את המפרשים של הדפדפנים הפופולרים:

Browser Interpreter
Google Chrome V8
Mozilla Firefox Gecko, SpiderMonkey, Rhino.
Safari Nitro
Microsoft Edge Chakra
Opera Carakan

הערות

כמו בכל שפת תכנות אחרת, אפשר להשתמש בהערות בקוד כך שהמפרש ידע להתעלם משורות אלו. פעמיים קו נטוי // - משתמשים להערה של שורה אחת. להערות של יותר משורה אחת נשתמש כמו בשפת C - נתחיל עם קו נטוי וכוכבית - /* ונסיים עם כוכבית וקו נטוי - */ 

                    
    // This line of code is commented and will be ignored by the web browser

    console.log('This statement will be executed');

    // console.log('This statement has been commented and will be ignored');

    /* 
        None of these lines
        will be executed
        by the interpreter
        console.log('This statement will be ignored');
    */

בסביבות עבודה יש קיצורי מקשים בשביל לסמן הערות. בVisual Studio Code אפשר להשתמש בCTRL + / בשביל הערה של שורה אחת. בשביל לסמן הערה של כמה שורות אפשר להשתמש בSHIFT + ALT + A

Case Sensitive

Javascript היא רגישה לאותיות קטנות או גדולות, מה שאומר: 

                    

    // This:

    console.log('something');

    // Is not the same as this:

    Console.log('something');

בדוגמא האחרונה תהיה אזהרה כיוון שאין אובייקט שנקרא "Console" עם C גדולה. בגלל זה, צריך להיות זהירים עם השמות של האובייקטים, מתודות, והגדרות כדי שלא נקבל שגיאות כאלו כתוצאה מטעות כזאת.

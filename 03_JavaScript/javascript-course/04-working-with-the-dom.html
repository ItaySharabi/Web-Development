עבודה עם הDOM

כדי ליצור אינטרקציה עם האלמנטים של HTML בדף שלנו נצטרך להשתמש במתודות ושדות של אובייקט המסמך.

getElementById and innerHTML

כדי לבחור אלמנט HTML נוכל לחפש אותם על ידי הclass שלהם, Id, שם הtag (h1, p, img, div, וכו) ועוד. נתחיל בId כיוון שזוהי הדרך הקלה ביותר.

לאלמנט הנוכחי יש "Id="blue_box. האם אתם יכולים לקחת את הטקסט בפנים ולשלוח לconsole? האם אתם יכולים לשנות את התוכן?

כדי לבחור בתיבה הכחולה נשתמש במתודה getElementByID ובשדה innerHTML. (זיכרו שJavascript היא רגישה לאותיות קטנות/ גדולות, לכן שימו לב מתי צריך להשתמש באות גדולה). 

                    
    
    // Getting the content of an element
    document.getElementById("blue_box").innerHTML;

    // Changing the content of an element
    document.getElementById("blue_box").innerHTML = "This will be the new content of the element.";

כעת בהתאם למה שראינו, ננסה לשלוח לconsole את התוכן שנמצא בתיבה הכחולה. לאחר מכן נשנה את תוכנה. 

                    
    
    console.log( document.getElementById("blue_box").innerHTML );
    document.getElementById("blue_box").innerHTML = "Some new content";

בתחילה אולי הדוגמא נראית פשוטה מדי אך זאת ההתחלה שבה מבינים איך לעבוד עם אלמנטים והתוכן שלהם.

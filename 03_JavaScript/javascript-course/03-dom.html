DOM - Document Object Model

כעת לאחר היכרות קצרה עם איך שJavascript עובדת, נלמד איך ליצור אינטרקציה עם אלמנטים בדף שלנו.

מה זה DOM?

הDOM הוא אובייקט שנוצר על ידי הדפדפן ברגע שדף נטען. האובייקט הזה מכיל את כל הקודקודים של דף הHTML שלנו, כלומר האלמנטים של הHTML. דוגמא פשוטה מופיעה להלן: 

                    
    
    <html>
        
        <head>
            <title>My Title</title>
        </head>
        
        <body>
            <h1>My Header</h1>
            <a href="#" >My Link</a>
        </body>

    </html>

כאשר הדף נטען, הדפדפן מייצר את עץ הDOM, כמו שנראה כאן:


בתמונה ניתן לראות שההורה אלמנט של כל שאר האלמנטים בDOM הוא אובייקט מסמך. כך שתמיד נשתמש באובייקט הזה כדי לבחור אלמנטים בדף שלנו.

בעזרת הDOM נוכל:

  • להוסיף, לשנות ולהסיר אלמנטים של HTML
  • לשנות את האטריביוטים של אלמנט HTML (כמו class או src של תמונה)
  • לשנות את הCSS
  • להגיב לאירועים בדף (לחיצה על העכבר, גלילה, קלט, וכו)

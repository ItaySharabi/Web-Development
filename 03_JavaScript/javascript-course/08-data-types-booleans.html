בוליאנים

בוליאנים הם הערכים 'אמת' או 'שקר'. הם נועדו לשלוט בכל ההיבט הלוגי בקוד שלנו.

בוליאנים נוצרים כאשר אנחנו משתמשים באופרטורי השוואה:

Operator Description
> Greater than
>= Greater than or equal to
< Less than
<= Less than or equal to
== Equal to
=== Identical (same type and value)
!= Not equal to
!== Not equal value or not equal type 
                    
    20 > 2; // true

    2 >= 2; // true

    5 < 5; // false

    5 <= 10; // true

    20 == 20; // true

    'john' == 'johnny' // false

    'a' == 'A' // false

    20 === "20"; // false - same value but different types

    20 != 10; // true

    10 != 10; // false

    "10" !== 10; // true - type is different


    /* 
    Be careful when using greater than and less than operators with different data types. 
    That can lead to unexpected results 
    */

נשתמש בבוליאנים בתנאים. לדוגמא, נאמר שתיתנו ללקוחות שלכם הטבות במידה וירכשו מוצר מעל 1000 דולר: 

                    
    var orderValue = 2800;

    if( orderValue >= 1000 ) {
        console.log("You are eligible to finance your order.");
    } else {
        console.log("Finance options are not available for orders below US$1000");
